# Personal CRM - Detailed Requirements

## User Stories

### Contact Management
- As a user, I want to see all my contacts in a clean, scrollable list
- As a user, I want to add new contacts quickly via a popup modal
- As a user, I want to edit contact information easily
- As a user, I want to categorize contacts by relationship type (family, friend, colleague)

### Frequency Scheduling
- As a user, I want to assign contact frequencies: weekly, monthly, quarterly, annually
- As a user, I want to see who I should contact today/this week
- As a user, I want to mark when I've contacted someone to reset their timer
- As a user, I want to see overdue contacts highlighted

### Birthday Management
- As a user, I want birthday reminders on the actual day
- As a user, I want a 1-week advance notice for potential card sending
- As a user, I want to see upcoming birthdays in the next 30 days
- As a user, I want to track gift ideas and what I gave previously

### Notes & History
- As a user, I want to add notes after conversations
- As a user, I want to see conversation history with timestamps
- As a user, I want to search through notes to find specific topics
- As a user, I want to tag notes for easy categorization

## Technical Requirements

### Mobile-First Design
- Touch-friendly buttons (minimum 44px tap targets)
- Swipe gestures for quick actions
- Optimized for one-handed use
- Fast loading and responsive interface
- Works offline with local data sync

### Data Privacy
- Local-first data storage
- Optional cloud backup with encryption
- No tracking or analytics
- User controls all data export/import

### Performance
- App loads in <2 seconds on 3G
- Smooth scrolling with large contact lists
- Efficient search and filtering
- Minimal battery usage
